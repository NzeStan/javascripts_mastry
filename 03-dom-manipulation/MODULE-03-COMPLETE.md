## â“ Common Questions

### Q: Should I memorize all DOM methods?
**A:** No! Focus on understanding. Reference docs when needed.

### Q: When should I use getElementById vs querySelector?
**A:** getElementById is faster for IDs, but querySelector is more flexible.

### Q: Is innerHTML dangerous?
**A:** Yes with user input! Use textContent for safety.

### Q: How do I know if I'm ready for Module 04?
**A:** If you can build the quiz from scratch (with notes), you're ready!

### Q: What if events aren't working?
**A:** Check that: 1) Element exists, 2) Listener is attached, 3) Script loads after HTML
