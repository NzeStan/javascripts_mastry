## VS Code Setup (Step-by-Step)

### Step 1: Install VS Code
1. Download from https://code.visualstudio.com/
2. Run the installer
3. Check "Add to PATH" during installation (important!)
4. Complete the installation

### Step 2: Install Essential Extensions
Open VS Code, then:
1. Click the Extensions icon (left sidebar) or press `Ctrl+Shift+X`
2. Install these extensions:

**Must-Have:**
- **Live Server** (by Ritwick Dey) - Run HTML files instantly
- **JavaScript (ES6) code snippets** - Code faster
- **Prettier - Code formatter** - Auto-format your code

---

## Browser DevTools Setup

### Opening DevTools
- **Chrome/Edge:** Press `F12` or `Ctrl+Shift+I`
- **Firefox:** Press `F12` or `Ctrl+Shift+I`

### Important Panels:
1. **Console** - Where you'll test JavaScript code
2. **Elements** - Inspect HTML/CSS
3. **Sources** - Debug JavaScript with breakpoints
4. **Network** - See API calls and resources

**Practice:** Open DevTools right now and type this in the Console:
```javascript
console.log("Hello, JavaScript!");
```
Press Enter. You should see the message!
