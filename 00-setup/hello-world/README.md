## How to Run This Project

### Method 1: Using Live Server (Recommended)
1. Open VS Code
2. Open the `javascript-mastery` folder
3. Navigate to `00-setup/hello-world/`
4. Right-click on `index.html`
5. Select "Open with Live Server"
6. Press `F12` to open DevTools Console

### Method 2: Double-Click
1. Just double-click `index.html`
2. It will open in your default browser
3. Press `F12` to open DevTools Console

## Understanding the Code

### HTML (index.html)
- Structure of your webpage
- Links to CSS and JavaScript files
- Contains a button and message paragraph

### CSS (style.css)
- Makes everything look beautiful
- Colors, spacing, animations

### JavaScript (script.js)
- Makes the page INTERACTIVE
- Responds to button clicks
- Changes content dynamically
- Logs messages to console

## Experiments to Try

1. **Change the student name** in `script.js`:
```javascript
let studentName = "Your Name Here";
```

2. **Change the button color** in `style.css`:
```css
button {
    background: lightgreen;  /* Try different colors! */
}
```

3. **Add more console logs** in `script.js`:
```javascript
console.log("I love JavaScript!");
```
